# EVSU Evaluation Survey System for the Registrar

This project is a web application built using **PHP** and **MySQL** for the Registrar's Office at Eastern Visayas State University - Ormoc Campus.

Follow the steps below to set up the project and run the development server locally.

---

## Requirements

Make sure the following are installed on your system:

- PHP 8.0 or newer
- MySQL 5.7 or newer
- Git
- XAMPP / WAMP / MAMP (recommended)

Check your PHP version:

```bash
php --version
